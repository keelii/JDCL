1. 插件名称：Jdropdown

3. 预览地址：[fiddle](http://jsfiddle.net/keelii/rRGC8/)、[jsBin](http://jsbin.com/EcawUmi/1/)

## 一、HTML 示例
html 代码示例

```html
<dl>
    <dt>标题 hover</dt>
    <dd>解释</dd>
    <dd>解释</dd>
    <dd>解释</dd>
</dl>
```

## 二、初始化脚本
```javascript
$('dl').eq(0).Jdropdown();
```

## 三、参数详情
* event —— 触发事件
* current —— 触发元素添加 class
* delay —— 延时
